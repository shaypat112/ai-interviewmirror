# AI Interview Mirror

Clean project structure:

- `frontend` - Next.js app for recording interview practice sessions
- `backend` - NestJS API for receiving and processing recordings

## Run the frontend

```bash
cd frontend
npm run dev
```

Frontend runs on `http://localhost:3000`.

## Run the backend

```bash
cd backend
npm run start:dev
```

Backend runs on `http://localhost:4000`.
